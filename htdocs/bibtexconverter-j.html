BibTeX converter: A Java GUI tool for BibTeX conversion

I' ve written an all-Java GUI application that converts BibTeX to BibXML and onwards.

For the first step the user can choose between my java port of your Python BibTeX parser and the sablecc-generated texlipse parser by Oskar Ojala. Both give good results on valid input, but the texlipse parser is faster and produces useful error messages.

The second step is done via XSLT. There are three built-in stylesheets (two HTML and one RIS) that use XSLT 2.0 features and don't work with the JRE default transformer factory implementation. Therefore the user is asked to install Saxon when the application first starts up.

Custom user-stylesheets can easily be loaded into the application (File menu). Unfortunately, I used a rather early BibXML DTD, so that one might need to modify your stylesheets a bit to make them run within this app (if I find the time I will maybe alter the program to use the latest BibTeXML schema).

BibTeXConverter needs Java 1.5 or higher. I put it under GPL 2.

The jar file can be run through 

java -jar bibtexconverter-20070223.jar
