BibTeX converter: A Java GUI tool for BibTeX conversion

BibTeX converter is an all-Java GUI application that converts BibTeX to BibXML and onwards.

For the first step the user can choose between a java port of the bibtexml.sf.net Python BibTeX parser and the sablecc-generated texlipse parser by Oskar Ojala. Both give good results on valid input, but the texlipse parser is faster and produces useful error messages.

The second step is done via XSLT. There are three built-in XSLT 2.0 stylesheets (two HTML and one RIS). Custom user-stylesheets can easily be loaded into the application (File menu).

BibTeXConverter needs Java 1.5 or higher und is published under GPL 2.

You can download the BibTeX converter source code with subversion from 

https://bibtexml.svn.sf.net/svnroot/bibtexml/trunk/bibtexconverter-j
An ant script is supplied for easy compilation.
